## Cpuinfo

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/cpuinfo.git
cd cpuinfo
go build -o cpuinfo .
```

### Usage

```bash
./cpuinfo --help
```

---

## Quick Start

```bash
./cpuinfo [command]
```

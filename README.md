## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/cpuinfo.git
cd cpuinfo
go build -o cpuinfo .
./cpuinfo --help
```

## 🚀 Quick Start

```bash
./cpuinfo -h
```

## 🧪 Examples

```bash
# Example 1
./cpuinfo status

# Example 2  
./cpuinfo --verbose

# Example 3
./cpuinfo help
```
